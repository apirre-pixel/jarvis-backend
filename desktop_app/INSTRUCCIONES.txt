   J.A.R.V.I.S Desktop v2 — Guía de instalación
═══════════════════════════════════════════════

NOVEDADES v2
────────────
✦ Icono en la bandeja del sistema (esquina inferior derecha)
✦ Voz Microsoft Neural (mucho mejor calidad)
✦ Configuración automática al primer arranque
✦ Menú con clic derecho sobre el icono


REQUISITOS
──────────
• Windows 10 / 11
• Python 3.9 o superior → https://www.python.org/downloads/
  ⚠ IMPORTANTE: marca "Add Python to PATH" al instalar


PASO 1 — Instalar dependencias
──────────────────────────────
Abre CMD o PowerShell y ejecuta:

  pip install SpeechRecognition edge-tts pygame requests pygetwindow pywin32 pystray Pillow

  Si pyaudio da error, instálalo así:
  pip install pipwin
  pipwin install pyaudio


PASO 2 — Ejecutar
──────────────────
En CMD, desde la carpeta donde están los archivos:

  python jarvis_desktop.py

La primera vez te pedirá:
  1. La URL de tu servidor en Render (ej: https://mi-jarvis.onrender.com)
  2. Tu clave GROQ_API_KEY (opcional, mejora el reconocimiento de voz)

Estos datos se guardan en config.json y no los volverá a pedir.


PASO 3 — Usar
──────────────
Di "Jarvis" → espera "Le escucho, Señor" → da tu orden.

Comandos de ventanas:
  "Jarvis, abre Opera"
  "Jarvis, abre Chrome"
  "Jarvis, cierra Spotify"
  "Jarvis, minimiza todo"
  "Jarvis, maximiza Discord"
  "Jarvis, cambia a WhatsApp"

Para todo lo demás habla con normalidad y la IA te responde.

ICONO EN LA BANDEJA
──────────────────
Aparece en la esquina inferior derecha (junto al reloj).
Clic derecho → opciones:
  • Pausar / reanudar micrófono
  • Editar configuración
  • Salir


CAMBIAR CONFIGURACIÓN MÁS ADELANTE
───────────────────────────────────
Clic derecho en el icono → "Editar configuración"
O edita config.json directamente con el bloc de notas.

Para cambiar la voz, edita config.json y cambia "voice":
  Hombre español:  "es-ES-AlvaroNeural"
  Mujer española:  "es-ES-ElviraNeural"
  Hombre latino:   "es-MX-JorgeNeural"
  Mujer latina:    "es-MX-DaliaNeural"


ARRANQUE AUTOMÁTICO CON WINDOWS (opcional)
──────────────────────────────────────────
1. Pulsa Win+R → escribe: shell:startup → Enter
2. Crea un acceso directo a jarvis_desktop.py en esa carpeta


SOLUCIÓN DE PROBLEMAS
──────────────────────
• pyaudio no instala    → pip install pipwin && pipwin install pyaudio
• "No module named..."  → pip install -r requirements.txt
• Sin respuesta Render  → Render free tier se duerme, espera 30s y reintenta
• Voz no suena          → comprueba que tienes conexión a internet (edge-tts la necesita)
